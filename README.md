# Email Replier Agent

## Overview
An AI-powered email response agent built using n8n. The workflow automatically generates email replies using an LLM without consulting external policy documents, terms and conditions, or knowledge bases. Responses are generated from the model's learned patterns and may contain hallucinations.

## Features
- Automated email reply generation
- LLM-powered responses
- n8n workflow automation
- Easy workflow import and deployment

## Technologies Used
- n8n
- AI Language Model
- JSON Workflow


## How to Use
1. Download the workflow.json file.
2. Import it into n8n.
3. Configure the required credentials.
4. Execute the workflow.
